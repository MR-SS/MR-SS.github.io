## 🚀 Deploy on GitHub Pages

### Option 1: User/Organization Site (`<username>.github.io`)

1. Create a new repository on GitHub named exactly: **`MR-SS.github.io`**
2. Clone it locally:
   ```bash
   git clone https://github.com/MR-SS/MR-SS.github.io.git
   cd MR-SS.github.io
   ```
3. Copy the files from this project into the repo:
   ```bash
   cp /path/to/landing-page/* .
   ```
4. Push to GitHub:
   ```bash
   git add .
   git commit -m "Initial commit: personal landing page"
   git push origin main
   ```
5. Your site will be live at: **https://MR-SS.github.io** 🎉

### Option 2: Project Site (any repo)

1. Create a new repository (e.g., `landing-page`)
2. Push `index.html` to a branch named `gh-pages`:
   ```bash
   git checkout -b gh-pages
   git add index.html
   git commit -m "Deploy landing page"
   git push origin gh-pages
   ```
3. Go to repo **Settings → Pages** and select the `gh-pages` branch
4. Your site will be live at: `https://MR-SS.github.io/landing-page`
